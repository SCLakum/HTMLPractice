List in HTML

Unordered HTML List

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. The list items will be marked with bullets (small black circles) by default.

Used list-style-type: default.

  • one
  • Two
  • Three

Used list-style-type: circle.

  • one
  • Two
  • Three

Used list-style-type: square.

  • one
  • Two
  • Three

Ordered HTML List

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag. The list items will be marked with numbers by default:

Used list-style-type: default.

  1. one
  2. Two
  3. Three

Used list-style-type: upper-alpha.

  1. one
  2. Two
  3. Three

Used list-style-type: asupper-roman.

  1. one
  2. Two
  3. Three

HTML Description Lists

HTML supports description lists, which consist of terms and their corresponding descriptions. The <dl> tag defines the description list, <dt> tags indicate the terms, and <dd> tags contain the descriptions.

  • Use the HTML < dl > element to define a description list.
  • Use the HTML < dt > element to define the description term
  • Use the HTML < dd > element to describe the term in a description list

Nested List

Nested lists in HTML refer to the practice of placing one list inside another list, creating a hierarchical structure. This allows for the organization of content with greater levels of detail. To create a nested list, you can use list elements such as < ul > (unordered list) or < ol > (ordered list) within another <li > (list item) element.
